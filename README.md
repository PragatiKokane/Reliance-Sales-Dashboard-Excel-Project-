# Reliance-Sales-Dashboard-Excel-Project-
Excel Dashboard Project using Pivot Tables, Charts, KPIs, and Slicers
📊 RELIANCE INDUSTRIES SALES PERFORMANCE DASHBOARD
📌 Project Overview

This dashboard analyzes the overall sales performance of Reliance Industries using Microsoft Excel. It provides insights into sales, profit, orders, product categories, sales channels, and regional performance through interactive KPIs, charts, and slicers.

✅ KPIs
Total Sales
Total Profit
Total Orders
Total Units Sold
Profit Margin %
📊 Charts Used
Monthly Sales Trend (Line Chart)
Region-wise Sales (Clustered Column Chart)
Product Category Sales (Bar Chart)
Sales Channel Contribution (Pie Chart)
Top 10 Products by Sales (Horizontal Bar Chart)
Sales vs Profit (Scatter Plot)
State-wise Sales (Map Chart)
🎛 Interactive Filters
Year
Region
State
Product Category
Sales Channel
